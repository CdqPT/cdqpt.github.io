# 一、内嵌CSS样式
```js
<p style="color: #AD5D0F;font-weight:bold;font-size: 30px; font-family: '宋体';" align="center">内联样式</p>
```
# 二、font-weight 属性
|值|描述|
|:---:|:---:|
|normal	|默认值。定义标准的字符。|
|bold|	定义粗体字符。|
|bolder	|定义更粗的字符。|
|lighter	|定义更细的字符。|
|100|定义由粗到细的字符。400 等同于 normal，而 700 等同于 bold。
|200|  同上
|300|同上
|400|同上
|500|同上
|600   | 同上        
|700|同上
|800|同上
|900 |同上
|inherit|	规定应该从父元素继承字体的粗细。|
