ubuntu下安装常用软件合集
linux
教程
发布日期:   2018-12-26
更新日期:   2020-02-25
阅读次数:  

一、安装谷歌浏览器

1.1 下载并安装google浏览器

Ubuntu版本Google浏览器下载链接：https://www.ubuntuchrome.com/

1.2 安装谷歌访问助手

谷歌访问助手链接：https://www.ggfwzs.com/

二、安装搜狗输入法

2.1 下载搜狗输入法Linux版软件包

下载地址为：https://pinyin.sogou.com/linux/ ,如下图，要选择与自己系统位数一致的安装包，我的系统是64位，所以下载64位的安装包。

2.2 更新软件库

先更新软件库

sudo apt-get update
sudo apt-get upgrade

切换目录

cd ~/Downloads/

2.3 安装软件包

sudo dpkg -i sogoupinyin_2.1.0.0082_amd64.deb

将sogoupinyin_2.1.0.0082_amd64.deb替换为安装包的名称。

2.4 安装依赖

sudo apt-get install -f

把系统键盘输入方式从ibus切换为fcitx，一般情况下默认就是fcitx，因此大多数情况下不用切换。

重启

打开fcitx config，取消仅当前语言，添加搜狗输入法。

将搜狗输入法拖到置顶
没有企鹅也是正常的

再次重启

三、安装日常办公软件

包括QQ,TIM,微信，百度网盘等。

3.1 安装deepin-wine环境

上https://github.com/wszqkzqk/deepin-wine-ubuntu页面下载zip包（或用git方式克隆），在“下载”目录下原地解压即可，

cd ~/Downloads/deepin-wine-ubuntu
sudo sh ./install.sh

3.2 安装deepin.com应用容器

在https://mirrors.aliyun.com/deepin/pool/non-free/d/中下载想要的容器，点击deb安装即可。以下为推荐容器:

  • QQ：https://mirrors.aliyun.com/deepin/pool/non-free/d/deepin.com.qq.im/
  • TIM：https://mirrors.aliyun.com/deepin/pool/non-free/d/deepin.com.qq.office/
  • QQ轻聊版：https://mirrors.aliyun.com/deepin/pool/non-free/d/deepin.com.qq.im.light/
  • 微信：https://mirrors.aliyun.com/deepin/pool/non-free/d/deepin.com.wechat/
  • Foxmail：https://mirrors.aliyun.com/deepin/pool/non-free/d/deepin.com.foxmail/
  • 百度网盘：https://mirrors.aliyun.com/deepin/pool/non-free/d/deepin.com.baidu.pan/
  • 360压缩：https://mirrors.aliyun.com/deepin/pool/non-free/d/deepin.cn.360.yasuo/

    3.3 安装.deb安装包

cd ~/Downloads/
sudo dpkg -i deepin.com.wechat_2.6.8.65deepin0_i386.deb
  -i后面替换成需要安装的.deb文件名。

然后去APP里搜索微信即可找到.

四、安装sublime

第一种方式

下载Linux安装包:https://www.sublimetext.com/3

得到:sublime_text_3_build_3200_x64.tar.bz2

解压:

tar -xvvf sublime_text_3_build_3200_x64.tar.bz2 sublime_text_3/
sudo mv sublime_text_3/ /opt

创建链接:

sudo ln -s /opt/sublime_text_3/sublime_text /usr/bin/sublime_text

创建图标:

ln -s /opt/sublime_text_3/sublime_text /usr/bin/sublime_text ~/Desktop

即可在桌面看到相应的快捷方式。这里前一个路径是程序的源路径，第二个是需要创建快捷方式的地方。

第二种方式

安装密匙

wget -qO - https://download.sublimetext.com/sublimehq-pub.gpg | sudo apt-key add -

确定用网址源下载

sudo apt-get install apt-transport-https

选择稳定版

echo "deb https://download.sublimetext.com/ apt/stable/" | sudo tee /etc/apt/sources.list.d/sublime-text.list

安装sublime

sudo apt-get install sublime-text
